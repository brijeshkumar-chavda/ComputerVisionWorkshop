# Tech Stack

## Front-end

1. Framework: React
2. Language: JavaScript
3. CSS: Vanilla CSS

## Back-end (Node.js)

1. Runtime: Node.js
2. Framework: Express.js (to create the API server)
3. AI SDK: openai (official NPM package, which works with Azure OpenAI)
4. File Uploads: multer (middleware to handle image/video file uploads)
5. Video Processing: fluent-ffmpeg & ffmpeg-static (to extract frames from uploaded videos server-side)
6. Utilities: cors (frontend-backend communication) and dotenv (security)
